# String Pickaxe

Fabric mod for Minecraft 1.21.11.

## Build

This project is pinned to the Java 21 install referenced by `java21.bat`.

```powershell
./gradlew build
```

The finished mod JAR is written to `build/libs/`.

If you move the Java 21 install, update `org.gradle.java.home` in `gradle.properties`.
